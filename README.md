## 项目说明
该项目主要用于检测ARP欺骗攻击, 能够检测到局域网中的ARP欺骗攻击并且显示出攻击者以及受害者的IP和MAC以便于定位受害者和攻击者
## 功能特性
1. Windows支持: 专为Windows系统设计, 使用Nacap驱动
2. 实时监听: 实时捕获网络接口上的ARP报文
3. 多维度检测: ARP报文内MAC与以太网帧源MAC不一致 -- IP-MAC映射关系变化 -- 网关ARP记录被篡改劫持
   
4. 智能判断: 自动推测本地网络网关地址
5. Npcap集成: 利用成熟的Npcap驱动进行高效包捕获
6. 友好交互: 支持网络接口选择, Ctrl+C安全退出
7. 高性能: 基于Rust和pnet库, 低资源占用
## 安装与使用
### 因为本工具需要依赖Nacap才能够正常使用, 所以需安装Nacap, Nacap的安装包和ARP_IDS.exe我会一起提供
## 注意事项
1. 必需管理员权限: Windows网络包捕获需要管理员权限
2. Nacap依赖: 必须安全Nacap驱动才能运行
3. 防病毒软件: 某些防病毒软件可能会阻止包捕获
4. 仅支持IPv4: 当前版本仅支持IPv4网络环境
5. 网关推测: 网关地址通过网段推算(通常为xxx.xxx.xxx.1), 可能与实际不符
